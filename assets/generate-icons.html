Generador de Iconos PWA - AcademiaOS

Este script genera iconos para la PWA de AcademiaOS en diferentes tamaños.

Instrucciones:

  1. Haz clic en "Generar Iconos" para crear todos los tamaños necesarios
  2. Descarga cada icono haciendo clic derecho y "Guardar imagen como..."
  3. Guarda los archivos en la carpeta assets/ con los nombres indicados
